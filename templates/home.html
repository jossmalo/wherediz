Wherediz



What is wherediz?

Wherediz provides a scriptable utility to determine the geographic location of an IP address. Simply pass an IP address using curl to get a response of the location.

Example:

$ curl wherediz.com/8.8.8.8
8.8.8.8: Mountain View, California, United States

Wherediz is designed to be script friendly if you need to get locations of a list of IPs, for example:

$ cat ip_list.txt
216.218.134.12
216.218.222.12
216.230.148.77
217.23.13.129
217.64.42.212
217.115.10.131
217.115.10.132
217.170.201.106

$ for i in $(cat ip_list.txt); do curl wherediz.com/$i ; done
216.218.134.12: Fremont, California, United States
216.218.222.12: Fremont, California, United States
216.230.148.77: Guatemala
217.23.13.129: Netherlands
217.64.42.212: Helsinki, Uusimaa, Finland
217.115.10.131: Germany
217.115.10.132: Germany
217.170.201.106: Oslo, Oslo County, Norway

Options

Currently wherediz only has a few options, to review them pass /help

$ curl wherediz.com/help
/[ip] - Shows the location of the IP Address
/json/[ip] - Provide a little more information in json format
/help - Show this message
/about - About this project
